# 多媒体

视频 `<video>` 和 音频`<audio> `两个标签


html多媒体使用到了html的`shadow-dom`


WebRTC:WebRTC中的RTC是实时通信的简称，这是一种支持在浏览器客户端之间语音/视频交流和数据分享的技术。WebRTC作为一项标准,使得所有浏览器无需安装插件或第三方软件，就可以点对点地分享应用数据和进行电话会议。
